# Personal Website — Daniyar (Danny) Abykhanov

Live: [personal-website-nu-three-23.vercel.app](https://personal-website-nu-three-23.vercel.app)

## Stack

- **Next.js 16** (App Router) + TypeScript
- **Tailwind CSS v4**
- **EmailJS** — contact form delivers directly to email
- Deployed on **Vercel**

## Sections

- **Hero** — typing animation, hackathon photo carousel
- **Projects** — TreeRoute, Knowledge Copilot, AI Investment Advisor
- **About** — bio, highlights, tech stack
- **Education** — M.S. Data Science @ University of Delaware
- **Experience** — work history timeline
- **Contact** — form with EmailJS integration

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
